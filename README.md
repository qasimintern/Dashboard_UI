**Employee Dashboard UI**

**Objective**

The goal of this project was to build a fully functional, modern, and responsive employee dashboard interface using HTML and CSS.
The dashboard includes the following key sections:

Dashboard Title

Key Metrics Cards 

Timesheet Table

Monthly Summary Section

Charts Integration Area

Responsive Design for All Devices


**HTML Structure (dashboard.html)**

Implemented a clean and semantic HTML layout.

Added sections for:

<header> – Dashboard title

<main> – Timesheet table, charts, and metrics cards


Structured all elements using meaningful class names for easy styling and chart integration.

**CSS Styling**

•	Created modern UI design with:

•	Gradient lines under section titles

•	summary cards

•	Clean, uniform tables for timesheet data

•	Consistent typography and spacing

•	Hover effects and shadows for improved UI feedback

•	Fully responsive layout for both mobile and desktop screens
